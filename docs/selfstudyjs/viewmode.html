On Click

  • 사용자와의 상호작용을 위한 Java Script
  • 웹 브라우저는 한번 화면에 출력되면 스스로 변경할 수 없다.(정적)
  • 자바스크립트는 HTML을 제어하는 언어이다.
  • 즉 자바스크립트는 HTML을 동적으로 만들어주는 언어라고 할 수 있다.

HTML 과 JS의 만남

  • script 태그
    스크립트 태그 안에 적힌 내용은 브라우저가 자바스크립트로 해석
  • 일반 태그의 자바스크립트 속성
    브라우저가 자바스크립트 관련 속성을 만나면 내용을 기억함, 이벤트가 일어나면 동작
    onclick, onchanged 등등
